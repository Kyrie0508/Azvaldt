## 개발의도
***
섀도우버스의 스토리를 이용한 게임을 만든다.<br> 
게임 방식은 RPG와 카드를 접목하여 섀도우버스를 <br>
즐기고 있는 사람들 뿐만 아니라, 섀도우버스를 처음 접하는 사람들도 쉽고 재밌게
접근 할 수 있도록 게임을 만드는 것이 목적이다. 
Unity를 통해 개발을 진행하도록 한다.
